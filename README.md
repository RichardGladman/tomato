#Tomato

Welcome to tomato, a work / break balance timer. It's not pomorodo as there are no longer breaks and the work / break periods are configurable.

It's easy to build, just follow the instuctions below.

First, you'll need to install [the qt framework](https://www.qt.io/development/download-qt-installer-oss)

Run the following commands:

```
git clone https://github.com/RichardGladman/project-timer.git
cd ansicodes
mkdir build
cd build
cmake ..
make
```

And optionally:

```
sudo make install
```
